Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assumble a dataset over 500 cities.

After assemlbing the data set, we used Matplotlib to plat various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidty. THis site provides the scource data and visualisations created as part of the analysis, as well as explainations and desciptions of any trends and correulations witnessed.
